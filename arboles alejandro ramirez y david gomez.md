
# Arboles con listas enlazadas

Este repositorio esta hecho por estudiantes de la UIS Alejandro ramirez y David gomez donde hacemos uso de listas Enlazadas para la creación de un árbol en phyton.
El código primero le pide al usuario que ingrese el valor de la raíz, una vez hecho esto aparece un menú donde pide ya sea añadir hijos al padre que desee el usuario o mostrar el peso, altura y orden del árbol.
